
# 📦 スタートアップ支援Webパック（1年契約・自動更新）

## 1. イニシャル（制作・初期設定費）：一律 100,000円

- **ここに含まれる価値（ここを強調！）：**

  - LP/HP制作

  - サーバー・ドメイン取得代行＆設定

  - GA4・タグマネージャー等の解析ツール設置（←これ素人無理だから価値高い！）

  - 問い合わせフォーム設置

## 2. ランニング（保守・管理費）：月額 20,000円

- サーバー・ドメイン維持費込み

- システム保守・監視

- 月1回程度の軽微な修正（画像差し替え、テキスト変更など）

- **「解約は2ヶ月前予告」**

## 3. 【オプション】広告運用代行：月額 30,000円〜

- Googleリスティング広告など

- 運用額の20%（ミニマム3万）

あとは基本的に今回のプランはアイデアベースなんですけど、イニシャルを一律10万円とかにしちゃうのを、本来であれば例えば30万以上、30から50万ぐらいが相場なんですけど、ここをエンドクライアント様の受注がしやすいようにランニングでここをカバーするっていうようなイメージのプランであることは、強調してあげるのが良いのかなと思っています。

僕が広告代理店なので、代理店もやりながらこういったデザインとかWEB制作を同時に一貫して一人の人間、僕が行えるっていうところが1つのメリットです。まるっと受注できることで安く提供できるのかなと思います。

そのランニングコストについては、相場が1万円から3万円の間というところだったので、平均値ぐらいの2万円にさせていただいて。

それでサーバー代とかドメイン代とかそういったところは、経費としてこの2万円の中に内包させてもらうイメージです。そうすればお客さんも30万や50万といった金額を払わずに、10万円でスタートできて、平均値ぐらいの保守管理費を払うというところがメリットになるのかなと思います。そうすればエンドクライアントからすると受注しやすいですし、新しく法人を立ち上げるような人たちであると仮定したら、助成金の対象経費にもなりやすいという点も魅力的です。こういう風に考えています。

その他かかる経費として、例えば予約フォームだったりとか、予約フォームの受信する機能とかは、外部のツールをサイトに埋め込んで、iFrameだったりで埋め込んで作成をするので、そこら辺も保守管理費に入れ込むのか、何なのかというイメージですね。
