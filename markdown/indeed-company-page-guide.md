# Indeed企業ページ 徹底活用ガイド

## はじめに

Indeed企業ページは、求人原稿だけでは伝えきれない「会社の魅力」を求職者に届ける強力なツールです。
このガイドでは、**活用方法**と**成果事例**にフォーカスし、企業ページを最大限に活かすための実践的な知識をまとめています。

---

## 1. Indeed企業ページとは

### 定義

Indeed企業ページ（Indeed Company Page）は、Indeed上で自社の文化・価値観・働く環境・福利厚生などをまとめて見せる「企業プロフィールページ」です。

- Indeed上に求人を出している企業は、**基本的に自動で企業ページが生成される**
- 企業側が「Claim（権限取得）」することで、内容を編集・管理できるようになる

### Free版とPremium版（CMPP）

| 項目 | Free版 | Premium版（CMPP） |
|------|--------|------------------|
| 会社概要・基本情報 | ○ | ○ |
| 写真・動画掲載 | ○ | ○（強化版） |
| 求人一覧との連携 | ○ | ○ |
| クチコミ・Q&A | ○ | ○ |
| ビジュアルの強化 | - | ○ |
| レビューのハイライト | - | ○ |
| 高度な分析機能 | - | ○ |

> **日本での注意点**
> - 「Employer Branding Hub」は日本では2025年時点で未実装
> - 日本で利用できるのは主に「企業ページプレミアム（CMPP）」

## 2. なぜ企業ページをやるべきか

### 求職者の行動変化

**求職者の67%が「オンライン上の企業クチコミが応募先選びに影響した」と回答**（Indeed調査）

求職者は今、仕事内容だけでなく：
- 企業文化
- 働く人のリアル
- 成長機会・柔軟性

を総合的に見て比較しています。

> **「求人原稿だけ」では、もはや候補者の判断材料として不十分。**
> **企業ページ ＝ 応募前に信頼と期待値を整える"前室"として機能している。**

### 世界の事例

#### Employer Branding Hub投資企業の平均値

| 指標 | 改善率 |
|------|--------|
| 求人クリック数 | **+27%** |
| 応募開始クリック数 | **+34%** |

| 指標 | 改善率 |

| 企業名検索ボリューム | **+200%** |

#### Featured Employer活用企業

- 企業ページのフォロワー数：半年で **+51%**
- ブランドインプレッション：数百万単位で増加

### 日本での期待効果

1. **認知度向上**
   - Indeed内での企業名検索時に露出増加
   - ブランド想起の向上

2. **求人閲覧数の増加**
   - 企業ページ経由で他の求人への回遊が発生
   - 求人詳細ページの閲覧回数UP

3. **応募数・応募率の改善**
   - 企業理解が深まることで「応募しても大丈夫そうだ」という心理ハードルが下がる

## 4. 企業ページプレミアム（CMPP）の活用

### 日本での利用条件

| 条件 | 目安 |
|------|------|
| 月の有料掲載額 | **最低150,000円程度以上** |
| その他 | Indeed側の基準（掲載形態・契約プランなど）を満たすこと |

> **ポイント**
> - CMPPは「別料金」というより、**毎月ある程度の広告費を出している企業への"無料オプション"**として付与されるイメージ
> - 最新の条件は **Indeedの営業担当または認定パートナー経由で確認**が安全

### CMPPで追加される機能

- **ビジュアルの強化**：バナー画像、動画、ストーリー的なコンテンツブロック
- **レビューのハイライト**：ポジティブレビューをピックアップ表示
- **高度な分析機能**：ページ閲覧数、ブランドインプレッション、評価推移など

## 5. 活用コンセプト

### 「応募数」だけでなく「応募の質」と「辞退率」までを見る

企業ページの本質的なゴールは：

- **「応募前に企業理解を深めてもらい、ミスマッチ応募と辞退を減らす」**
- **「自社とフィットする人だけを強く惹きつける」**

単純な応募件数だけを追う施策ではありません。

### 「求人原稿」と「企業ページ」をセットで設計

| 役割 | 内容 |
|------|------|
| **求人原稿** | ポジション単位での「仕事の中身」「条件」「要件」 |
| **企業ページ** | 会社単位での「なぜここで働くのか」「どんな人が活躍しているか」 |

> **求人原稿の「ロジック」と、企業ページの「ストーリー」を組み合わせることで、候補者の納得感を最大化する。**

## 6. コンテンツ設計のベストプラクティス

### 会社概要でおさえる要素

- **事業ドメイン**：何を提供している会社か
- **顧客**：誰に価値を届けているか
- **強み**：他社と違うポイント
- **ミッション／ビジョン**：中長期でどこを目指すか
- **バリュー**：行動規範

### 写真・動画コンテンツ

おすすめ：
- オフィス・現場の写真（盛ってない"日常感"を重視）
- メンバーインタビュー動画（30〜90秒程度）
- 1日の流れをまとめたショート動画
- 代表・現場リーダーのメッセージ動画

> **ポイント**：「キラキラしすぎない」「実態に近い」トーンにすることで、入社後ギャップを減らせる。

### クチコミ＆Q&A対応

- ネガティブレビューも含めて、**放置せずコメントする**（反論ではなく、事実と改善姿勢を示す）
- Q&Aには**24〜48時間以内**の返答を目安に
- 「よくある質問」はテンプレ化して回答品質を揃える

## 7. 成果を最大化する運用フロー

### STEP 0：企業ページをClaimして基礎を埋める

- 企業ページをClaimし、基本情報・ロゴ・写真を埋める
- レーティング推移やレビューのモニタリングを開始

### STEP 1：ページ改善 → 求人原稿へのフィードバック

企業ページ上の分析から、求人原稿のコピーや訴求軸にフィードバック：

| 発見 | アクション |
|------|-----------|
| 「教育体制が良い」というレビューが多い | 原稿の"教育・研修"訴求を強める |
| 「ワークライフバランス」が評価されている | 残業時間・休暇実績を数字で見せる |

### STEP 2：効果検証のKPI設計

Before / Afterで追うべき指標：

**ブランド指標**
- 企業名検索数（Indeed内）
- 企業ページ閲覧数
- フォロワー数・レビュー数の推移

**求人行動指標**
- 企業ページ経由の求人クリック数
- 応募開始クリック（apply start）の数・率

**採用成果指標**
- 応募数／有効応募数
- 面接設定率・内定率
- 入社後早期離職率（中長期）

## 8. よくあるNG・注意点

| NG例 | 問題点 |
|------|--------|
| 作って終わりで情報が古いまま | 「止まっている会社」に見える |
| クチコミを完全放置 | 「何もケアしていない会社」という印象 |
| 自社サイト・求人票とのメッセージ不一致 | 信頼が削られる |
| "盛りすぎ"のブランディング | 入社後ギャップ → 早期離職に直結 |

## 9. まとめ

### 無料でも期待できる効果

- 認知度向上
- 求人閲覧数増加
- 応募数・応募率の改善

### CMPPで期待できる追加効果

- ブランドインプレッション：+20〜30%
- 求人クリック・応募開始クリック：+20〜30%

### 成果を出すための核心

**「求人原稿 × 企業ページ × 広告運用」をワンセットで設計し、**
**応募数ではなく"応募の質"と"辞退・離職の低減"まで含めて見ること。**

## 補足：日本でのアナリティクス確認方法

### 企業アカウントから確認できる指標

日本のIndeed管理画面でも、企業アカウントから以下の手順で企業ページ関連の指標を確認できます：

**アクセス手順**
1. Indeedに企業アカウントでログイン
2. 左メニューの **「アナリティクス」** をクリック
3. 上部タブの **「Employer Branding Ads」** を選択

**確認できる指標（指標の推移）**

| カテゴリ | 指標 |
|----------|------|
| **合計** | 求人の表示回数 |
| | 求人のクリック数 |
| | 企業ページの閲覧数 |
| | クチコミの閲覧数 |
| | 動画の視聴回数 |
| | 企業のフォロワー数 |
| | 企業ページでの求人の表示回数 |
| | 企業ページでの求人のクリック数 |
| **コンバージョン率** | 求人のクリック率（CTR） |

> **ポイント**
> - これらの指標は **企業アカウントのみ** から閲覧可能
> - 日・週・月単位で推移を確認できる
> - 「データの絞り込み」で期間やキャンペーンを指定可能
